# 🌉 Izmirim-Wallet-Bridge (PoC)
### "Fiziksel Karttan Dijital Cüzdana Akıllı Köprü"

Bu proje, akıllı şehir ulaşım sistemlerinde kullanıcı deneyimini iyileştirmek ve atıl finansal kaynakları ekonomiye kazandırmak amacıyla geliştirilmiş bir **Kavram Kanıtı (Proof of Concept)** çalışmasıdır.

## ❓ Problem Nedir?
İzmir ulaşımında kullanılan fiziksel İzmirim Kartlar ile mobil uygulama üzerindeki dijital kartlar iki ayrı cüzdan gibi çalışmaktadır. Bu durum:
- Fiziksel karttaki bakiyenin dijitalde kullanılamamasına,
- Kullanıcıların "yetersiz bakiye" stresi yaşamasına,
- Milyonlarca kartta küçük ama toplamda devasa bir "atıl bakiye" birikmesine neden olmaktadır.

## ✅ Çözüm: Unified Wallet (Birleşik Cüzdan)
Bu MVP (Minimum Uygulanabilir Ürün), fiziksel karttaki bakiyeyi NFC veya Kart ID aracılığıyla saniyeler içinde dijital cüzdana aktarmayı sağlar.

### 🛠 Özellikler
- **Hibrit Giriş:** NFC destekli cihazlar için anlık okuma, diğer cihazlar için manuel ID girişi.
- **Modüler Mimari:** İş mantığı (Logic) ve sürdürülebilirlik metrikleri ayrıştırılmış temiz kod yapısı.
- **ESG Odaklı Yaklaşım:** Atıl kaynakların sisteme dahil edilerek finansal sürdürülebilirliğin artırılması.

## 💻 Teknik Kurulum
1. Projeyi klonlayın: `git clone https://github.com/KULLANICI_ADIN/Izmirim-Wallet-Bridge.git`

3. Çalıştırın: `python main.py`
